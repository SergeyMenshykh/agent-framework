# PIN Strength Guide

## Requirements

| Use Case | Min Length | Example |
|----------|-----------|---------|
| Device unlock | 6 | `847291` |
| High-security | 8 | `39174826` |

## Rules

- PINs must not contain repeating digits (`1111`).
- PINs must not contain sequential runs (`1234`, `4321`).
- Minimum length is 4 digits.
